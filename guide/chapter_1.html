Feakin - 软件开发工业化 DSL

DDD in Feakin

Feakin 的 DDD 语法参考了：https://contextmapper.org/docs/context-map/

Feakin 语法可以实现三个阶段：

  • 战略设计。语法：ContextMap、Context
  • 战术设计。语法：Aggregate、Entity、ValueObject
  • 实现。语法：impl

Quick Start

  1. 在线编辑器：https://online.feakin.com/
  2. 在 Template -> DDD Booking 使用 Feakin 语法设计
ContextMap TicketBooking {
    Reservation -> Cinema;
    Reservation -> Movie;
    Reservation -> User;
}

Context Reservation {
  Aggregate Reservation;
}

Context Cinema {
  Aggregate Cinema;
}

Aggregate Cinema {
  Entity Cinema, ScreeningRoom, Seat;
}

DDD Strategy

Step 1. Design Context Map

表示领域上下文图，用于描述系统的边界。

ContextMap TicketBooking {
  Reservation -> Cinema;
  Reservation -> Movie;
  Reservation -> User;
}

关系：

  • - - Undirected,
  • -> - PositiveDirected,
  • <- - NegativeDirected,
  • <-> - BiDirected,

Step 2. Context

表示限界上下文，以及对应的聚合间的关系。

Context Reservation {
  Aggregate Reservation;
}
